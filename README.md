# School-Management-System-cpp
A console-based School Management System developed in C++ for Programming Fundamentals.
## Features
- Admin login
  - Add student records
  - View all records
  - Search, edit, and delete student records
- Student login
  - View own record
  - Edit own record
  - Cannot delete records (security-based access)

## Technologies Used
- C++
- File Handling
- Structures
- Conditional Statements
- Loops

## How to Run
1. Compile the file:
   g++ record.cpp -o record
2. Run:
   ./record
